Oto-Sepet

The Oto-Sepet project has been developed using the C# programming language. This application is specifically designed for use by car rental companies. Oto-Sepet provides users with various features such as adding and deleting vehicles, as well as daily, weekly, and monthly rental rates. Developed to simplify and manage car rental processes, this application aims to be an effective tool in the industry. The project's goal is to carry out vehicle rental operations more efficiently and user-friendly, while also assisting rental companies in better managing their daily operations.

BMI-Calculator

BMI-Calculator is a mobile application developed using Flutter. This app provides a tool for users to measure their health by performing BMI (Body Mass Index) calculations. BMI Calculator calculates the body mass index based on the user's height and weight, offering general information about their health status.
